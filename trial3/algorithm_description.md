| **Algorithm 1** &nbsp; Classification of Linear Codes over $PG(k-1, q)$ with Symmetry Breaking |
|:---|
| **Input:**<br>&nbsp;&nbsp;&nbsp;&nbsp;$n$: Length of the code<br>&nbsp;&nbsp;&nbsp;&nbsp;$k$: Dimension of the code<br>&nbsp;&nbsp;&nbsp;&nbsp;$q$: Size of the finite field $\mathbb{F}_q$<br>&nbsp;&nbsp;&nbsp;&nbsp;$W$: Set of target non-zero weights $\{w_1, w_2, \dots, w_m\}$<br><br>**Output:**<br>&nbsp;&nbsp;&nbsp;&nbsp;$\mathcal{C}_{unique}$: A set of non-equivalent $[n, k]_q$ linear codes satisfying $W$<br>---<br>**Procedure:**<br><br>1. &nbsp; **Initialization (Geometry Generation)**<br>    &nbsp;&nbsp;&nbsp;&nbsp;Generate points $\mathcal{P}$ and hyperplanes $\mathcal{H}$ for projective geometry $PG(k-1, q)$.<br>    &nbsp;&nbsp;&nbsp;&nbsp;Compute incidence matrix $M$ where $M_{ij} = 1$ if $P_j \in H_i$, else $0$.<br><br>2. &nbsp; **Phase 0.5: Theoretical Pruning (Fail Fast)**<br>    &nbsp;&nbsp;&nbsp;&nbsp;**if** GriesmerBound($n, k, q$) is violated **then** return $\emptyset$<br>    &nbsp;&nbsp;&nbsp;&nbsp;**if** PlessPowerMoments($n, k, q, W$) are not satisfiable **then** return $\emptyset$<br><br>3. &nbsp; **Phase 0: Feasibility Check (Relaxed ILP)**<br>    &nbsp;&nbsp;&nbsp;&nbsp;Construct a relaxed ILP model (continuous or simplified constraints).<br>    &nbsp;&nbsp;&nbsp;&nbsp;**if** Model is Infeasible **then** return $\emptyset$<br><br>4. &nbsp; **Phase 1.5: Symmetry Analysis (Orbital Branching)**<br>    &nbsp;&nbsp;&nbsp;&nbsp;Compute the automorphism group $G \cong PGL(k, q)$ acting on $\mathcal{P}$.<br>    &nbsp;&nbsp;&nbsp;&nbsp;Partition $\mathcal{P}$ into orbits $\mathcal{O} = \{O_1, O_2, \dots, O_r\}$ under $G$.<br>    &nbsp;&nbsp;&nbsp;&nbsp;Select representative points $p_i \in O_i$ for each orbit.<br><br>5. &nbsp; **Phase 1: Exact Search (Branch-and-Cut)**<br>    &nbsp;&nbsp;&nbsp;&nbsp;$\mathcal{S}_{candidates} \leftarrow \emptyset$<br>    &nbsp;&nbsp;&nbsp;&nbsp;**for** each orbit representative $p_i$ **do**:<br>        &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Construct ILP model with binary variables $x_j$ (or integer for multisets).<br>        &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Add constraints: $\forall H \in \mathcal{H}, \sum_{P_j \in H} x_j \in \{n - w \mid w \in W\}$.<br>        &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Add symmetry breaking constraint: Force $x_{p_i} = 1$ (fix the first point).<br>        &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Solve using Gurobi Optimizer.<br>        &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;$\mathcal{S}_{candidates} \leftarrow \mathcal{S}_{candidates} \cup \text{FoundSolutions}$<br>    &nbsp;&nbsp;&nbsp;&nbsp;**end for**<br><br>6. &nbsp; **Phase 2: Verification & Filtering**<br>    &nbsp;&nbsp;&nbsp;&nbsp;$\mathcal{S}_{verified} \leftarrow \{ C \in \mathcal{S}_{candidates} \mid \text{VerifyWeightDistribution}(C, W) \}$<br>    &nbsp;&nbsp;&nbsp;&nbsp;$\mathcal{C}_{unique} \leftarrow \text{FilterIsomorphic}(\mathcal{S}_{verified})$ using code equivalence check.<br><br>7. &nbsp; **Return** $\mathcal{C}_{unique}$ |
